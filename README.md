# Superstore Sales Dashboard

## Project Overview
This project analyzes Superstore sales data using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## Dashboard Features
- Sales and Profit by Region
- Sales and Profit by Category
- Top 10 Products by Sales
- Monthly Sales Trend
- Sales by Customer Segment
- Profit by Sub-category

## Key Insights
- West region generates the highest sales and profit.
- Technology is the most profitable category.
- Canon Copier is the highest-selling product.
- Sales peak in November.
- Consumer segment contributes the highest sales.
- Copiers generate the highest profit.
